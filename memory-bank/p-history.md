## P-001 — 初始策略提示词（2026-04-05）

**意图**: 每6小时扫描所有目标城市天气市场，发现模型概率与市场价格的偏差

**生成器版本**: G v0.1

**提示词内容**:
```
扫描目标城市列表中的所有活跃天气市场。
对每个城市：
1. 获取最新 GFS + ECMWF + UKMO + NWS 预报
2. 多模型加权融合，计算每个温度区间的概率分布 P_model
3. 获取 Polymarket 该城市各温度区间的 YES 价格 P_market
4. 计算 Edge = P_model - P_market
5. 过滤: |Edge| > 5% AND spread < $0.05 AND 距结算 > 2h
6. 对通过过滤的信号，Quarter-Kelly sizing（上限 15% bankroll）
7. 风控检查全部通过后，提交限价单
8. 记录信号和执行结果到数据库
```

**状态**: 待验证（Phase 2 完成后首次运行）
