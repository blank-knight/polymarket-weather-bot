# 生成器 G — 迭代规则

> 每次 M 更新后写入新版本

## v0.1 — 初始规则（2026-04-05）

来源: `strategy-research.md` 中的经验值

### 策略选择规则
- 主策略: 模型更新时间差套利（每6小时执行）
- 辅助策略: 时间衰减收割（结算前12h执行）
- 长尾策略: 价格 ≤ $0.05 但模型概率 ≥ 5% 时小额下注

### Edge 规则
- 最小 Edge: 5%
- 理想 Edge: 15-30%
- Edge = P_model - P_market
- 正 Edge → BUY YES; 负 Edge → BUY NO

### Kelly Sizing 规则
- 使用 Quarter-Kelly (f*/4)
- 单笔上限: 15% bankroll
- 单笔下限: $1
- f* = (p*b - q) / b

### 退出规则
- 止盈: 60% Edge 被捕获
- 追踪止损: 从峰值回撤 40%
- 止损: -15%
- Edge 收敛退出: 剩余 Edge < 2%
- 时间退出: 结算前 2 小时

### 模型权重
- ECMWF: 0.35
- GFS: 0.25
- UKMO: 0.20
- NWS: 0.20
- 离群值降权: 偏离共识 > 2σ → 0.5x

### 模型置信度（标准差）
- 6h 预报: σ = 0.8°C
- 1d 预报: σ = 1.5°C
- 3d 预报: σ = 2.5°C
- 7d 预报: σ = 4.0°C
- 10d+ 预报: σ = 5.5°C

### 风控规则
- 单市场最大仓位: 15% bankroll
- 总敞口上限: 50% bankroll
- 日亏损上限: 10% bankroll
- 最低流动性: bid/ask spread < $0.05
- 结算前保护: 2h 内不开新仓
